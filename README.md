# autoinfra

**Code anywhere. Deploy everywhere. Visualize everything.**

> A lightweight, industrial-grade AI framework in C++ with a Keras-like Python API and dedicated IDE.

## Features

- Modern C++17/20 core
- Full training and inference support
- Multi-backend (CPU, CUDA, OpenCL)
- Python API with Keras-style syntax
- Cross-platform (Windows, Linux, macOS)
- MIT License

## Quick Build

```bash
git clone https://github.com/Viethedev/autoinfra.git
cd autoinfra
mkdir build && cd build
cmake ..
cmake --build .
